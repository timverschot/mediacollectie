Prijstracker

Collectie Prijzen Beheer

Prijsrange en gemiddelde zijn gebaseerd op actieve eBay-vraagprijzen (geen bevestigde verkoopprijzen), opgehaald via scripts/price_tracker.py --refresh. Elke keer dat je dat script draait, komt er een nieuw punt bij in de trendlijn hieronder.

Nog geen prijsdata. Draai python scripts/price_tracker.py --refresh om te starten.
